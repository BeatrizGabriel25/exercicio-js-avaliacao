\# Exercise 2 - Ecommerce Homepage



\## Description



This exercise is an Ecommerce homepage built with \*\*plain HTML, CSS, and JavaScript\*\*.  

It demonstrates:



\- Loading product data from an API (\[Fake Store API](https://fakestoreapi.com/products))

\- Dynamically creating a \*\*grid of products\*\* with their images and names

\- Adding an \*\*"Add to Cart" button\*\* for each product

\- Sending a \*\*PUT request\*\* to update a cart at `https://fakestoreapi.com/carts/7`

\- Displaying a \*\*dynamic footer\*\* with the current year



\## How to Run



1\. Open `index.html` in a web browser.

2\. The products will load automatically.

3\. Click \*\*"Add to Cart"\*\* on any product to send it to the API.

4\. The footer shows the current year automatically.



\## Technologies Used



\- HTML5

\- CSS3

\- JavaScript (ES6)

\- Fetch API



\## Notes



\- The page is fully client-side, all logic is in the `index.html` file.

\- This exercise demonstrates asynchronous requests, DOM manipulation, and dynamic content generation.
